# Vue.js

## Vue 설치

- sudo npm install -g vue
- sudo npm install -g @vue/cli



## Vue3 프로젝트 생성

- vue create [project name]
- Vue3 선택
- npm run serve
- cd [project name]
- vue add vuetify
  - vuetify 2 => 오류
  - vuetify 3 => 정상
- v3-vutify3-test.zip
